# Project name

Random quotes generator.

## git clone

Use the package manager [git](https://github.com/bpinazmul18/npm-rand-quotes-gen) to install foobar.

```bash
git clone https://github.com/bpinazmul18/npm-rand-quotes-gen.git
```

## Usage

```nodejs
node index
```
